
# Humbleicons

![icon](humbleicons.svg?raw=true)

## Usage

Depending on your setup, you can include Humbleicons in these ways:

- Copy-paste SVGs as embedded HTML
- Reference via `<img>` element
- Use the SVG sprite
- Include via CSS
